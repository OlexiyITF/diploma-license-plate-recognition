# 🚗 Система оптичного розпізнавання автомобільних номерних знаків (ALPR)

Програмний комплекс для автоматизованого розпізнавання тексту на українських номерних знаках. Проєкт розроблено в рамках дипломної роботи на факультеті прикладної математики та інформатики.

## 📌 Основний функціонал
* **Гібридна архітектура CRNN:** Використання згорткових (CNN) та рекурентних (BiLSTM) шарів із функцією втрат CTC для динамічного розпізнавання номерів змінної довжини (від 3 до 8 символів).
* **Базова модель ResNet18:** Класична багатошарова архітектура (Multi-Head) для розпізнавання стандартних номерних знаків.
* **Парсинг даних:** Автоматизований збір навчальної вибірки реальних фотографій для формування власного унікального датасету (`parse_autoria.py`).
* **Веб-інтерфейс:** Інтерактивний додаток на базі Streamlit для тестування моделей (Inference) у реальному часі.

## 📂 Структура репозиторію
* `app.py` — головний файл веб-інтерфейсу (GUI) для взаємодії з моделями.
* `train_crnn.py` — скрипт конвеєра навчання гібридної мережі (CRNN + CTC).
* `train_resnet.py` — скрипт навчання базової моделі класифікації.
* `parse_autoria.py` — утиліта для автоматизованого збору зображень.
